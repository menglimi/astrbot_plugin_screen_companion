# 屏幕伴侣插件更新日志

## 2.2.0 (2026-03-07)

### 新增功能
- **自定义预设系统**：用户可以通过指令配置多个自定义互动预设
- **快速预设切换**：通过 `/kpi [数字]` 快速切换不同预设
- **预设指令配置**：通过 `/kpi y [序号] [间隔] [概率]` 快速设置预设

### 改进
- **移除预设模式**：删除了轻度/中度/高频/静默等预设模式，只保留自定义模式
- **运行时动态切换**：在自动化任务运行过程中可以实时切换预设，任务会在下次判定时自动应用新参数
- **参数即时生效**：切换预设后，如果新间隔比当前等待时间短，任务会提前触发
- **索引安全保护**：添加了完善的边界检查，防止索引越界导致的崩溃

### 指令更新
- `/kpi y 1 90 30` - 设置预设1：90秒检查一次，30%概率触发窥屏
- `/kpi 1` - 切换到预设1
- `/kpi -1` - 切换到手动配置模式
- `/kpi presets` 或 `/kpi p` - 查看所有自定义预设

### 新增功能
- **电量感知增强**：优化了电池状态检测功能，提供更精确的电量信息和充电状态提示
- **自我学习系统**：增强了学习能力，能够从用户的反馈和行为模式中更智能地调整互动方式
- **记忆优化**：改进了长期记忆系统，实现了记忆衰减、关联和优先级管理，提供更个性化的互动体验
- **智能场景识别**：增强了场景识别能力，能够更准确地识别用户当前的活动和环境
- **多模态支持**：优化了视觉和语言模型的集成，提供更丰富的互动方式

### 改进
- **性能优化**：优化了截图和图像处理流程，减少了系统资源占用
- **稳定性提升**：修复了多个潜在的崩溃和错误情况，提高了插件的稳定性
- **用户体验优化**：改进了交互流程和反馈机制，提供更流畅的用户体验
- **配置灵活性**：增加了更多可配置选项，让用户可以更精细地调整插件行为
- **文档更新**：更新了插件文档，提供更详细的使用指南和配置说明

## 2.0.1 (2026-03-07)

### 核心改进
- **状态管理系统**：实现了三种状态（活动、非活动、临时任务）的管理
- **任务冲突修复**：解决了任务无法结束的问题，确保任务能够正确取消和清理
- **日记格式优化**：修复了日记中重复标题和日期的问题
- **日记图片优化**：增加了日记图片的高度，确保内容显示完整
- **临时任务改进**：确保临时任务不会触发活动状态，保持状态的一致性
- **代码结构优化**：重写了部分代码，提高了可靠性和可维护性

## 2.0.0 (2026-03-05)

### 重大更新
- **框架人格支持**：现在可以使用 AstrBot 框架的默认人格了，插件会优先使用 `persona_manager` 获取人格
- **人格统一**：所有 LLM 调用现在都统一使用相同的人格设定
- **超时优化**：增加了各个 API 调用的超时时间
- **日记图片优化**：改进了中文显示和图片宽度
- **任务取消修复**：修复了 `kps` 命令无法正确结束任务的问题
- **提示词优化**：简化了所有提示词，保持简短
- **配置更新**：新增 `system_prompt` 配置项，统一了配置默认值

## 1.9.0 (2026-03-04)

### 新增功能
- **调试模式**：添加了调试模式开关，可通过 `/kpi debug [on/off]` 命令控制
- **日记补写**：添加了 `/kpi complete [YYYY-MM-DD]` 命令，可以补写指定日期的日记

### 改进
- **简化命令**：简化了 KPI 子指令，将 `/kpi recent-diaries` 简化为 `/kpi recent`
- **平台适配**：修复了 l2d 桌宠平台的主动消息发送问题
- **日志优化**：更新了日志系统，使其在调试模式关闭时更加简洁
